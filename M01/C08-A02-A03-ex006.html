Bold

Using the B tag, we have bold formatting (not semmantic)

Using the STRONG tag, we have highlight formatting (semmantic)

Italic

Using the I tag, we have italic formatting (not semmantic)

Using the EM tag, we have emphasis formatting (semmantic)

Marked

Using the MARK tag, we have marked text formatting

Big and Small Text

Using the BIG tag, we have big text formatting (deprecated)

Using the SMALL tag, we have small text formatting

Deleted and Inserted Text

Using the DEL tag, we have a text that can be read but not considered

Using the INS tag, we have a text that was inserted

Underline

Using the U tag, we have underline formatting (not semmantic)

Subscript and Superscript Text

Using the SUB tag, we have a subscript formatting

Using the SUP tag, we have a superscript formatting
